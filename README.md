# 📱 TNB - Temporary Number Bot

## 🌐 Live Demo
[View Demo](https://adminx007.github.io/TNB/)

## ✨ Features

### User Panel
- 📊 Dashboard with statistics
- 📱 Get temporary numbers
- 🔑 Receive OTP codes
- 🎁 Referral program
- 👤 Profile management

### Admin Panel
- 📈 Overview dashboard
- 👥 User management
- 📱 Number management
- 🔑 OTP code tracking
- 💳 Withdrawal management
- ⚙️ Settings configuration
- 💾 Backup & restore

## 🚀 Tech Stack
- Firebase (Auth, Firestore)
- HTML5, CSS3, JavaScript
- Chart.js for analytics
- PWA ready

## 📦 Installation

### Firebase Setup
1. Create a Firebase project
2. Enable Email/Password authentication
3. Set up Firestore database
4. Update `firebase-config.js` with your credentials

```bash
# Clone repository
